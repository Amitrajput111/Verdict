# Verdict V2

**The Participation Network** – a social debate app where users watch content, pick a side, react, comment, earn streaks, and build a role identity.

## Tech Stack
- **Frontend**: Expo (React Native), React Navigation, Zustand, Socket.io‑client
- **Backend**: Express.js, Socket.io, mock in‑memory DB (Postgres/Redis compatible)
- **Deployment**: Vercel (static Expo web build + Serverless Functions for the API)

## Getting Started (local)
```bash
# Clone repo (already done)
npm install               # install dependencies
npm run start             # Expo dev server (web: http://localhost:8082)
```

## Production Build & Deploy
The repo contains a `vercel.json` configuration. Deploy is automated via the Vercel CLI:
```bash
npx vercel --prod --confirm --token <YOUR_VERCELE_TOKEN>
```
The command builds the Expo web bundle (`web-build/`) and deploys the backend as a serverless function.
